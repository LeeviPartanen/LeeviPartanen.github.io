IoT & Robotics

Surveillance System

In this project, I built a surveillance system for a Raspberry Pi using a PIR sensor and a camera. When the PIR sensor detects motion, the program automatically starts video recording. The system includes a web interface where users can view the camera feed in real time and stop the recording manually. The recording is also set to stop automatically after 30 minutes if the user is not present. Video recordings are stored on the Raspberry Pi. The project uses Python and Flask for the web interface.

Energy Management Dashboard

This React + Vite application displays the current electricity market price and plots a graph of electricity prices over the course of the day. The example application uses mock data for electricity prices but can easily be connected to a real API.
